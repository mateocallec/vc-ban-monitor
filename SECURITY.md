# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in VC Ban Monitor, please report it responsibly via email:

**Contact:** mateo@callec.net

Do not create public issues for security vulnerabilities. Please allow at least 72 hours for a response before disclosing publicly.

## Supported Versions

This policy applies to all current and future versions of VC Ban Monitor.

## Security Best Practices

- Always keep Chrome updated.
- Only download the extension from official sources.
- Do not share your personal credentials in the extension or with third-party scripts.
